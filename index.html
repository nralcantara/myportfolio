MY PROJECTS

Click on the images to learn more about the website

FlyN8

FLYN8 is a static website which was created with HTML and Bootstap. Bootstrap is a CSS Framework for developing responsive and mobile-first websites.

This website was created to demonstrate a simple product promotion for a travel and tours agency.

FRAME

FRAME is a dynamic website which was created with LARAVEL Framework. A PHP web framework created by Taylor Ottwell and intended for development of web applications following the Model-View-Controller (MVC) architectural pattern.

This website was created to demonstrate an Asset Management System of a rental store using the operations of Create, Read, Update and Delete (CRUD) wherein the admin user is able to add, display, edit and delete products from and to MySQL Database.

RoadTrip

ROADTRIP is a full-stack website which was created with a Client-Server Model. This is a distributed application structure that partitions tasks between provider (server) and service requester (clients). A basic example of a microservices web architecture.

This website was created to demonstrate a bus transport Booking System using MongoDB, Express, React and Node applications.
